2.1.1 Page Layout and Content

The layout of my page is very simplistic with each unit having its own page which is accessible via the tabs at the top of each page. Each tab is clearly labeled. I chose bright colours with white text in order to make the content stand out and legible but also quite calming.
I have gone for the same look on all the pages for a uniform look across the site. THis is important to me for the sake of consistency. All content on the page is centered for a more centralised look on the site. The text on the page has been well spaced out so it doesn't look word heavy and intimidating

2.1.5 Copyright Constraints

“Copyright refers to the legal protection of original works of authorship. This includes literary, written, dramatic, artistic, musical, and other types of creative expression.
Websites contain written material, graphics, business logos, and other visual and sound elements. This firmly categorizes them as copyrightable material.”
When building a website, it is important to be careful what material is used in the content of the site so as not to be liable for copyright infringement.⁷

2.1.6 Access Issues

Someone who has a vision impairment or sight disability may struggle to use the site I am building as I haven’t optimised it for such a person. Likewise someone who is hard of hearing, although there is no music on my site. In order to cater for said disabilities, I would need to make sure the site has an option where the characters can be enlarged/superimposed to allow for ease of reading.

2.1.7 Appropriate File Types

There is a long list of internet file types, depending on what is to be included in the building of the site. Some of the most popular are MPEG for video, MP3 for music, JPEG for images etc. I will be using the following for my site.

  • HTML - HyperText Markup Language

    • "HTML is a computer language devised to allow website creation. These websites can then be viewed by anyone else connected to the Internet."

  • CSS — Cascading StyleSheet

    • "CSS files are a tool in the repertoire of webmasters that take care of how their websites look."

  • JPEG — Joint Photographic Experts Group file

    "A very common image file format, mainly used for photos"⁸
